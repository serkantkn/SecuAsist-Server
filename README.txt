SecuAsist Sunucu Kurulumu (Windows)
==================================

1. Bilgisayarınızda Python yüklü olduğundan emin olun (Python 3.10+ önerilir).
2. Bu "Server" klasörünü bilgisayarınızın herhangi bir yerine kopyalayın.
3. Klasör içindeki "baslat.bat" dosyasını çift tıklayarak çalıştırın.
4. Dosya ilk açıldığında gerekli kütüphaneleri (FastAPI, Websockets vb.) internetten indirecektir.
5. Sunucu başarıyla başladıktan sonra tabletlerinizden sunucu IP adresine bağlanabilirsiniz.

Not: Web arayüzüne tarayıcınızdan "http://localhost:8000" adresinden erişebilirsiniz.
